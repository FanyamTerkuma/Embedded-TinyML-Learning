# Phase 1: Embedded Basics with Simulators

## Goal

Write real embedded code without owning any hardware. Wokwi is your virtual lab.

## Timeline and Schedule (10 Weeks)

| Period | Focus | Key Tasks | Milestone |
|---|---|---|---|
| Wk 1-2 | Arduino Core in Wokwi | Blink LED, button-controlled LED, traffic light simulation. Learn digitalWrite, pinMode, delay, Serial monitor. | 3 working Wokwi projects |
| Wk 3-4 | Sensors and Analog I/O | Potentiometer + LED brightness, LM35 temperature sensor, photoresistor. Learn analogRead, PWM and analogWrite. | Analog sensor project live |
| Wk 5-6 | ESP32 + Peripherals | ESP32 + OLED display (I2C), simple web server simulation, DHT22 temperature sensor. Learn I2C, WiFi basics, libraries. | ESP32 project with display |
| Wk 7-8 | Interrupts and Timers | Button with interrupt (no debounce), millis() timing instead of delay(). Learn attachInterrupt, why delay() is harmful for responsiveness. | Non-blocking blink + counter running simultaneously |
| Wk 9-10 | Capstone + PlatformIO | Build Smart Plant Monitor (temperature + moisture + OLED + buzzer). Switch from Arduino IDE to VS Code + PlatformIO. | 6+ projects on GitHub + PlatformIO configured |

## Expected Outcome

6+ working simulated projects on GitHub with Arduino and ESP32.
