# Phase 1: Embedded Basics with Simulators

This phase introduces hands-on embedded programming using software simulators before working with real hardware.

## Topics

- Introduction to Arduino and AVR microcontrollers
- GPIO: digital input/output, pull-up/pull-down resistors
- Timers, delays, and interrupts
- Serial communication (UART)
- Using simulators: Wokwi, Tinkercad, Proteus
- Reading datasheets and reference manuals
- Debouncing, state machines, and simple sensor interfacing

## Goals

By the end of this phase you should be able to write and simulate embedded firmware that reads sensors, controls outputs, and communicates over UART — all inside a simulator before touching real hardware.
