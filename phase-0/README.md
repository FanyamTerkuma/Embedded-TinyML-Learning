# Phase 0: Foundations - Electronics & C

## Goal

Understand electricity and C so you are not lost later. Think of this as installing your embedded brain.

## Timeline and Schedule (5 Weeks)

| Period | Focus | Key Tasks | Milestone |
|---|---|---|---|
| Week 1 | Electronics Part 1 | Study V/I/R, Ohm's Law, resistors, LEDs, breadboards (electronics-tutorials.ws sections 1-4). Solve 15 quiz questions and draw 5 circuits on paper. | Explain why a 220 ohm resistor is needed with an LED |
| Week 2 | Electronics Part 2 + Digital Logic | Capacitors, switches, logic gates, binary (sections 5-7 + digital logic). Paper exercises on truth tables. | Understand pull-up and pull-down resistors |
| Week 3 | C Programming - Core | learn-c.org lessons 1-6 (variables, loops, functions, arrays). Write 10 small programs on Replit. | Comfortable with loops and arrays |
| Week 4 | C Programming - Embedded Style | learn-c.org lessons 7-10 (pointers, bitwise, structs). 30 min intro to C++ classes. Bit-manipulation exercises. | Program using pointers and bitwise operations |
| Week 5 | Review + Mini Project | Simulate an LED blink with timing in an online C compiler (pretend it is firmware). Commit first GitHub repo with all notes. | Phase 0 GitHub repo live |

## Expected Outcome

Comfortable with C, basic electronics, and the language of embedded systems.
