# Phase 0: Foundations — Electronics & C

This phase covers the foundational knowledge required for embedded systems development.

## Topics

- Basic electronics: voltage, current, resistance, Ohm's Law
- Digital logic: gates, flip-flops, binary/hex number systems
- Circuit analysis and schematic reading
- Introduction to microcontrollers and their architecture
- C programming fundamentals: data types, operators, control flow
- Pointers, memory layout, and manual memory management in C
- Bitwise operations and bit manipulation
- Makefiles and the GCC toolchain

## Goals

By the end of this phase you should be comfortable reading simple schematics, writing C programs from scratch, and understanding how software maps to hardware registers.
