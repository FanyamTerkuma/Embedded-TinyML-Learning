# Phase 5: Integration, Portfolio & Hardware

This final phase brings everything together into complete, portfolio-worthy embedded systems projects.

## Topics

- System architecture and hardware/software co-design
- Integrating custom PCBs with TinyML inference pipelines
- Wireless connectivity: BLE, Wi-Fi, LoRa, MQTT
- Cloud connectivity and edge-to-cloud data pipelines
- Power budgeting and battery-powered system design
- Enclosure design and mechanical integration
- Documentation: technical write-ups, schematics, firmware READMEs
- Building a public portfolio (GitHub, Hackaday, personal blog)

## Goals

By the end of this phase you should have at least one end-to-end embedded + TinyML project that demonstrates hardware design, firmware development, and AI inference — fully documented and ready to showcase.
