## Goal

Tie everything together into a compelling portfolio, then jump to real hardware.

| Period | Focus | Key Tasks | Milestone |
|---|---|---|---|
| Wk 1-2 | Full End-to-End Integration | Take phase 3 PCB + phase 2 firmware + phase 4 TinyML model into one master repo. Simulate end-to-end in Wokwi with a clean architecture diagram. | Master repo with all components integrated |

Complete portfolio plus first real hardware project.
