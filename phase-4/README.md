# Phase 4: AI / TinyML on Embedded

This phase focuses on deploying machine learning models on resource-constrained microcontrollers.

## Topics

- Introduction to TinyML and edge AI concepts
- Data collection and dataset preparation for embedded applications
- Training models with TensorFlow / Keras
- Model optimization: quantization, pruning, and knowledge distillation
- Converting models with TensorFlow Lite and TFLite Micro
- Deploying inference on microcontrollers (Arduino Nano 33 BLE, STM32, etc.)
- Edge Impulse end-to-end workflow
- Keyword spotting, image classification, and anomaly detection use-cases

## Goals

By the end of this phase you should be able to train, optimize, and deploy a neural network model that runs inference on a microcontroller with limited memory and compute resources.
