# Phase 4: AI / TinyML on Embedded

## Goal

Train and deploy machine learning models that run on microcontrollers.

## Timeline and Schedule (10 Weeks)

| Period | Focus | Key Tasks | Milestone |
|---|---|---|---|
| Wk 1-3 | TinyML Fundamentals | HarvardX Fundamentals of TinyML on edX (free audit, active in 2026). Complete first 3 modules (2-4 hrs/week). Learn ML basics and neural nets on tiny devices. | Modules 1-3 of edX course done |
| Wk 4-6 | Edge Impulse Platform | Edge Impulse Getting Started + Arm Learning Paths audio classifier guide. Projects: keyword spotting (yes/no), gesture recognition using phone accelerometer data. | Two trained models: voice + gesture |
| Wk 7-8 | Deploy to Simulator | Paste Edge Impulse generated C++ library into Wokwi ESP32 project. Test live inference in browser and record accuracy in README. | TinyML inference running in Wokwi |
| Wk 9-10 | TinyML Capstone | Build Voice-controlled Plant Monitor or Anomaly detection on vibration data. Combine phase 3 PCB design + phase 2 firmware + TinyML model in one repo. | Full simulated TinyML system documented |

## Expected Outcome

Full simulated TinyML system running on an ESP32.
